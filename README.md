1.What is the difference between var, let, and const?
১নং প্রশ্নের উত্তর 
var হলো Function scoped, re-declare করা যায়।
let হলো Block scoped, re-declare করা যায় না, value change করা যায়।
const হলো Block scoped, re-declare ও re-assign দুটোই করা যায় না।



2.What is the spread operator (...)?
২নং প্রশ্নের উত্তর 
Array বা Object এর element ছড়িয়ে কপি করার অপারেটর।

3.What is the difference between map(), filter(), and forEach()?
৩নং প্রশ্নের উত্তর 
map নতুন array return করে
filter শর্ত অনুযায়ী array return করে
forEach শুধু loop, কিছু return করে না।


4.What is an arrow function?
৪নং প্রশ্নের উত্তর 
Function লেখার ছোট ও modern syntax।

5.What are template literals?
৫নং প্রশ্নের উত্তর 
Backtick ব্যবহার করে string এর ভিতরে variable বসানোর একটা পদ্ধতি।
